ENGLISH:

Usage:

It’s pretty simple. Just add the lines

set as_enable=true
set as_user=(Your Last.fm Username)
set as_pass=(Your Last.fm Password)

to the cmus config file. Or just type them as ":set " commands in cmus.

Toggling listening statistics reporting off/on is accomplished by ":set as_enable=false"/"set as_enable=true" respectively.

*******************************************************************

TÜRKÇE:

Kullanım:

Çok kolay. Sadece aşağıdaki satırları cmus ayar dosyanıza ekleyin:

set as_enable=true
set as_user=(Your Last.fm Username)
set as_pass=(Your Last.fm Password)

ya da ":set " yazan komutları cmus'ta yazın.

Dinleme istatistiklerinizi LastFm'e gönderme işlemini açıp kapamayı ise ":set as_enable=false"/"set as_enable=true" komutlarıyla sağlayabilirsiniz.
